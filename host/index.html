📖 Como funciona o sistema binário?

O sistema binário (base 2) representa números usando apenas dois dígitos: 0 e 1 (bits).


Cada posição em um byte representa uma potência de 2.



Para converter binário → decimal, some as potências onde o bit for 1.


Exemplo: 11010110

128+ 64+ 32+ 16+ 8+ 4+ 2+ 1= 214

Com 8 bits (1 byte) podemos representar números de 0 a 255.
